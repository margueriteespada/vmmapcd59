Utilisation de vMap et du module Cadastre
===================

.. toctree::
   :maxdepth: 2
   
vMap est une application de webmaping permettant l’affichage de cartes géographiques sur un navigateur.
La présente documentation est la manuel d'utilisation de l'application. Il décrit la navigation sur les carte ainsi que le module cadastre destiné à la consultation grapique et attributaire des données cadastrales.
